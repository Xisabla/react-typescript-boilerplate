# React Typescript Boilerplate

Ready to go react typescript template with webpack

**Note**: It uses *yarn*

```bash
git clone https://github.com/Xisabla/react-typescript myproject
cd myproject
yarn install
yarn start
```

## Technologies

- Basic
  - Typescript
  - React
- Styling
  - Sass
  - Autoprefixer
- Assets
  - Fonts importing
  - Images importing
- Linting
  - Eslint
  - Prettier
  - Stylelint


## Recipes

- `start`: clean, lint, build and start with http-server
- `build`: build with webpack
- `dev`: devserver with webpack and hot module replacement
- `clean`: remove all built files
- `lint`: run eslint and stylelint with fix options
